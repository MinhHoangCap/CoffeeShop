# CoffeeShop
Web for Coffee Shop

Description: Web được tạo ra để có thể xem được menu các loại thức uống và phần contact có thể email được cho quán coffee.

Frontend: Sử dụng Reactjs
    Sử dụng: 
        cd ./vite-project : để di chuyển tới thư mục vite-project
        yarn dev : để chạy frontend
Backend: Sử dụng Nodejs để kết nối database để tạo ra restAPI để kết nối frontend để show ra thông tin sản phẩm
    Sử dụng:
        cd ./server : để di chuyển tới thư mục server
        npm start : để run code backend
Database: Sử dụng MongoDB để chứa dữ liệu
